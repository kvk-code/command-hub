### Official KTU Curriculum vs. NSSCE Implementation

There exists a notable discrepancy between:

1. **KTU Official Curriculum Codes**

   - May use standardized codes across all affiliated colleges

2. **NSSCE Internal Course Codes**
   - Used internally for course management
   - May be customized for NSSCE-specific implementation
   - `CSD416` is the code actually used at NSSCE

### Why Multiple Codes Exist

**Possible Reasons:**
- **Curriculum Updates:** Codes may have changed between syllabus revisions
- **Department Variations:** Different departments/colleges may use different codes
- **System Migrations:** Changes in academic management systems
- **Manual Entry Errors:** Faculty using outdated reference materials
- **Phase Confusion:** Different codes for Project Phase I vs Phase II
